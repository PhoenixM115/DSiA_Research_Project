# DSiA_Research_Project
This project contains data cleaning, processing, visualization, and statistical analysis of crisis response calls in Eugene and Springfield, Oregon. The project focuses on how Mobile Crisis Services of Lane County (MCSLC) response times and call characteristics changed following its rollout in August 2024.

The analysis examines multiple stages of the response pipeline, including request to dispatch, dispatch to arrival, and arrival to engagement. It also compares changes in dispatch reason, disposition, city, and agency response patterns where CAHOOTS and EPD data are available.

The repository is organized into two main parts:

- `cleaning/`: contains data cleaning and preparation notebooks
- `analysis/`: contains visualization and statistical analysis notebooks

For more details, see the individual README files in the `cleaning` and `analysis` folders.
